## Installation

```bash
$ npm install -g print-cli
```


<a id="markdown-usage" name="usage"></a>
## Usage

```bash
$ print-cli --help
$ print-cli --version
```

<a id="markdown-updating-print-cli" name="updating-print-cli"></a>
## Updating print-cli

Global package:
```bash
$ npm uninstall -g print-cli
$ npm cache clean
$ npm install -g print-cli@latest
```
